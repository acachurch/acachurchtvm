## ğŸ› ï¸ Development

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd acachurchtvm

# Install dependencies
npm install
```

### Running the Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## ğŸ¨ Customization

### Updating Church Information

All church-related content is centralized in `src/config/churchInfo.js`. Update this file to change:

- Church name, address, contact details
- Service times
- Events
- Pastor information
- Ministries
- Social media links

### Changing Colors

The color scheme is defined using CSS variables in `src/index.css`. Modify the `:root` section to change the theme colors.
