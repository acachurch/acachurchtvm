# Church Images

## How to Add Images

### Pastor Photo
1. Place the pastor's photo in `src/assets/images/pastor/`
2. Name it `pastor.jpg` (or `.png`, `.webp`)
3. Update `churchInfo.js` to use: `pastorImage`

### Gallery Photos
1. Place gallery photos in `src/assets/images/gallery/`
2. Name them descriptively (e.g., `sunday-worship.jpg`, `youth-fellowship.jpg`)
3. Update `churchInfo.js` gallery array with the filenames

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp` (recommended for best performance)

## Tips
- Use descriptive filenames (lowercase, hyphens instead of spaces)
- Optimize images before adding (compress to reduce file size)
- Recommended sizes:
  - Pastor photo: 800x1000px (portrait)
  - Gallery photos: 1200x800px (landscape)
